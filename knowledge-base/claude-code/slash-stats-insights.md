## /stats, /insights, /release-notes — מידע וניתוח

פקודות לצפייה בסטטיסטיקות, ניתוח sessions והיסטוריית שינויים.

---

## /stats — סטטיסטיקות שימוש

מה זה עושה: ויזואליזציה של שימוש יומי והיסטוריית סשנים.

```bash
/stats
```

### מה מקבלים
- גרף שימוש יומי ב-tokens
- מספר sessions פעילים
- היסטוריית שימוש לאורך זמן

---

## /insights — ניתוח סשנים

מה זה עושה: מנתח וסוכם סשנים קודמים — דפוסי שימוש, משימות שחזרו, טיפים לשיפור.

```bash
/insights
```

### מתי להשתמש
- רוצה להבין איך אתה משתמש ב-Claude Code
- מחפש דפוסים חוזרים שאפשר לייעל
- סקירה שבועית/חודשית

---

## /release-notes — היסטוריית עדכונים

מה זה עושה: מציג changelog מלא של Claude Code — כל העדכונים, תכונות חדשות ותיקונים.

```bash
/release-notes
```

### מתי להשתמש
- אחרי עדכון גרסה — לראות מה חדש
- כשתכונה עובדת אחרת מהצפוי — אולי השתנתה
- לבדוק אם פיצ'ר שאתה מחכה לו כבר זמין
