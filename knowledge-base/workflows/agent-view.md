`claude agents` פותח מסך אחד לניהול כל הסשנים הרקע שלכם: מה שרץ, מה ממתין לקלט, ומה הסתיים.

## מה זה עושה / למה זה שימושי

Agent view מאפשר לשלוח מספר משימות עצמאיות ל-Claude בו-זמנית, לעקוב אחר כל אחת בזמן אמת, ולהתערב רק כשנדרש. כל סשן רקע ממשיך לרוץ גם כשאינכם מחוברים אליו — הוא מנוהל על ידי תהליך supervisor נפרד.

```bash
claude agents
```

## איך משתמשים

### פתיחת Agent View

```bash
claude agents
```

לסינון לפי פרויקט ספציפי:

```bash
claude agents --cwd ~/projects/my-app
```

### שליחת משימה חדשה

הקלידו prompt בתיבת הקלט בתחתית ולחצו Enter. כל prompt שמוכנס פותח סשן חדש עצמאי — לא שולח הודעת המשך לסשן קיים. ניתן לשלוח מספר prompts ולקבל ריצה מקבילה.

### קריאת מצב סשנים

כל שורה בטבלה מציגה מצב עם אייקון:

| מצב | אייקון | משמעות |
| :--- | :--- | :--- |
| Working | מונפש | Claude מריץ כלים או מייצר תגובה |
| Needs input | צהוב | ממתין לשאלה או לאישור |
| Idle | עמום | מוכן לפרומפט הבא |
| Completed | ירוק | המשימה הסתיימה בהצלחה |
| Failed | אדום | הסתיים עם שגיאה |
| Stopped | אפור | הופסק ידנית |

צורת האייקון מציינת אם התהליך עצמו פעיל (`✻`) או כבוי (`∙`). סשן כבוי עדיין ניתן לחיבור — Claude יאתחל מנקודת הפסקה.

### Peek ותגובה

בחרו שורה עם מקשי החצים ולחצו `Space` לפתיחת חלון peek — מציג את הפלט האחרון של הסשן או השאלה שהוא ממתין לה. הקלידו תגובה ולחצו Enter בלי לעזוב את agent view.

### חיבור וניתוק

לחצו `Enter` או `→` על שורה לחיבור לסשן המלא. הסשן כובש את הטרמינל כסשן Claude Code אינטראקטיבי. לחצו `←` על prompt ריק לחזרה לטבלה.

### העברת סשן קיים לרקע

```text
/bg
```

או לחצו `←` על prompt ריק בתוך סשן אינטראקטיבי.

## ניהול מ-shell

```bash
# רשימת סשנים בפורמט JSON
claude agents --json

# עצירת סשן
claude stop <session-id>

# מחיקת סשן
claude rm <session-id>
```

## הרצת פקודת shell כסשן רקע

```bash
claude agents
```

בתוך agent view: `! <command>` מריץ פקודת shell כסשן רקע שניתן לחבר אליה ולנתקה.
